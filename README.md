# Landing Page

Projeto de uma **landing page responsiva** desenvolvido com foco em prática de layout moderno, componentização e organização de interface para aplicações web.

## 🚀 Tecnologias utilizadas

- **Angular**
- **TypeScript**
- **Tailwind CSS**
- **HTML**
- **SCSS**

## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de aprimorar habilidades em desenvolvimento front-end, utilizando o ecossistema do Angular para criar uma página moderna, responsiva e visualmente agradável.

A aplicação foi construída pensando em:

- estruturação de componentes
- boas práticas de organização de código
- responsividade para diferentes dispositivos
- estilização moderna com Tailwind CSS
- tipagem e escalabilidade com TypeScript

## 🎯 Objetivo

Este projeto faz parte do meu processo de evolução como desenvolvedor front-end e será utilizado na composição do meu **portfólio**, demonstrando meus conhecimentos na criação de interfaces com Angular.

## 📱 Responsividade

A landing page foi desenvolvida para oferecer uma boa experiência de navegação em diferentes tamanhos de tela, como:

- desktop
- tablet
- smartphone
